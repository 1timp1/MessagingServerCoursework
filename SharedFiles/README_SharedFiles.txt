This is the SharedFiles folder.
Place any files you want to share with clients here.
Clients can download these files using the /DOWNLOAD command.

Supported file types:
- Text files (.txt, .md, etc.)
- Images (.jpg, .png, .gif, etc.)
- Audio files (.mp3, .wav, etc.)
- Video files (.mp4, .avi, etc.)
- Any other file type
